This PRD is a **compile-time type-system / module-boundary** change. The relevant evidence forms are **grammar-fixture** (anti-mismatch), **wired-on-main / anti-orphan** (the bug sites + the reuse machinery + the real migration consumers), **field-population** (the carried `type_expr` field has a real producer), and **rejection-backed** (Task C's negative-assertion).
